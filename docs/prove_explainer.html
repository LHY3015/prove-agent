Procedural Reuse via Outcome-Verified Executables

PROVE — procedural memory as executable skills, lifecycle governed by objective outcomes.

A skill is written from verified experience, recalled instead of re-prompted, and forgotten once it stops working. Each of those steps follows a deterministic downstream check, never a model's opinion — and attribution decides which memory is at fault, so a healthy skill is never dropped for a failure it did not cause.
